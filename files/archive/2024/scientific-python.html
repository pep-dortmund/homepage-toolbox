Fitten

Oft mchte man eine Funktion mit freien Parametern, zum Beispiel eine Erwartung aus der Theorie, an die gemessenen Werte anpassen. Dies nennt man Fit.

Die Funktion scipy.optimize.curve_fit nutzt die numerische Methode der kleinsten Quadrate, die arbitrre Funktionen fitten kann. Fr Funktionen, die eine Linearkombination von Einzelfunktionen sind, also

$$ f(x) = \sum_i^N a_i \cdot f_i(x) $$

existiert eine analytische Lsung. Deswegen sollten in solchen Fllen (z.B. alle Polynome) entsprechende Funktionen genutzt werden (z.B. np.polyfit).

Nichtlineare Funktionen der Parameter

Wenn eine Funktion nicht linear bezglich der freien Parameter ist, muss die Lsung numerisch gefunden werden.

Hierbei kann es sein, dass der Minimierungsalgorithmus in lokale Maxima hineinluft und unsinnige Ergebnisse liefert, dies kann mit guten Startwerten meistens vermieden werden.

Als Beispiel einer komplexeren Funktion wird im Folgenden die Sigmoidfunktion verwendet (hnlich zum tanh):

$$ f(x; a, b, c) = \frac{a}{1 + \exp(-(x-b))} + c$$

In[9]:
def sigmoid(x, a, b, c):
    return a / (1 + np.exp(-(x - b))) + c

Die richtigen Startwerte findet man entweder durch

  1. trial and error => einfach ausprobieren bis es klappt

  2. nachdenken => siehe unten

Im obigen Beispiel musste nur der Parameter b angepasst werden, weil der fr die Form der Kurve sehr wichtig ist.

$$ f(x; a, b, c) = \frac{a}{1 + \exp(-(x-b))} + c$$

Statistische Verteilungen und Funktionen
